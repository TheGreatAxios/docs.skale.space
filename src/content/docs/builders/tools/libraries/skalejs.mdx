Some of the available packages are:
- [@skaleproject/ima](https://www.npmjs.com/package/@skaleproject/ima): provide content to help using IMA, the SKALE native bridge
- [@skaleproject/pow-ethers](https://www.npmjs.com/package/@skaleproject/pow-ethers): SKALE proof of work mechanism written with Ethers library
- [@skaleproject/pow-web3](https://www.npmjs.com/package/@skaleproject/pow-web3): SKALE proof of work mechanism written with web3 library
- [@skaleproject/utils](https://www.npmjs.com/package/@skaleproject/utils): package with utility scripts, such as, interaction with SKALE access_control predeployed contract
