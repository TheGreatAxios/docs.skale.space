Crossmint allows anyone to purchase NFTs using any payment method, even without a wallet or cryptocurrency.

Buyers receive the NFTs directly in their wallet if they have one. If not, Crossmint will create a wallet on-the-fly, which can be embedded in your site or accessed via crossmint.com.

## Features

- **Wallet Solutions:** Crossmint wallets can be configured for custodial or non-custodial signing, with customizable permissions to meet diverse security needs. Each project includes a dedicated MPC vault and a gas station for seamless fee management, eliminating the complexities of crypto treasury operations.  

- **NFT Minting Tools:** Offers powerful APIs and no-code tools to mint, edit, airdrop, and burn NFTs at scale.

- **NFT Checkout:** Powers fast and reliable NFT checkout experiences, trusted by major marketplaces. Allows users to buy or claim NFTs directly through customizable storefronts or claim pages.
