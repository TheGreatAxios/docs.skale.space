## Features

- **Smart Wallet:** Sequence provides a non-custodial smart wallet with multi-chain support, offering users a smooth and secure way to interact with blockchain apps and assets.  

- **Developer-Friendly Platform:** Comes with a developer dashboard that makes it easy to quickly interact with different Sequence features (ex: smart contract deployment, marketplace, etc).

- **Smart Contract Deployment:** Simplifies deploying and managing smart contracts with developer-friendly tools.  

- **Whitelabel Marketplace:** Enables seamless creation of customizable marketplaces tailored to project needs.  

- **Contracts API:** Provides an easy-to-use API for interacting with deployed smart contracts programmatically.
