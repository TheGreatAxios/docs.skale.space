Crossmint allows anyone to purchase NFTs using any payment method, even without a wallet or cryptocurrency.

Buyers receive the NFTs directly in their wallet if they have one. If not, Crossmint will create a wallet on-the-fly, which can be embedded in your site or accessed via crossmint.com.
