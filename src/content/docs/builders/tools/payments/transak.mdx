Transak On-Ramp solution allows users to buy crypto assets with fiat.

The Transak On-Ramp integration allows users to purchase Europa USDC with fiat.

## Features

- **On-Ramp:** Purchase USDC on SKALE Europa Hub with credit card

- **Embeded On-ramp Widget:** Transak SDK integration on different platforms (ex: web, mobile, etc)

## Quickstart

To begin using Transak:

<Steps>

1. Go to the [SKALE Portal](https://portal.skale.space/)

2. On the left side select [On-Ramp](https://portal.skale.space/onramp)

3. Select the amount you want to pay

4. Choose the payment method

5. Select **Buy Now**

</Steps>
