Web3.unreal is a plugin that brings web3 functionality to desktop games built with Unreal Engine.

## Features

- **EVM compatibility:** Provides interaction with all EVM based chains as well as its smart contracts.

- **Hyperplay:** Direct Hyperplay integrations.
