## Features

- **EVM compatibility:** Provides interaction with all EVM based chains as well as its smart contracts.

- **Develepment platforms:** The SDK supports both Unity and Unreal.

- **Wallets:** Integrates WalletConnect

- **Platforms:** Supports mobile, webGL and standlone projects.
