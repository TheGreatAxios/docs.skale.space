Reservoir APIs offer comprehensive endpoints for developing NFT applications.
The NFT data APIs provide granular token price data, real-time collection floor price and top bid events, token and collection metadata, and more.

## Features

- **User Data:** Provides detailed collection metadata, statistics, and market data for collections owned by a user

- **Events:** Tracks events by contracts

- **Execute Order:** Has available APIs for purchase and sell NFTs 

## Quickstart

<Steps>

1. Head to the Reservoir [dashboard](https://dashboard.reservoir.tools/) to get an API Key 

2. Check the available endpoints for SKALE Nebula [here](https://docs.reservoir.tools/reference/nebula-nft-api).

3. Start using the NFT API.

</Steps>
