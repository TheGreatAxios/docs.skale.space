Dripverse allows the easy creation of NFTs, collections as well as trade them on their market. 
Integration and usage of these utilities can be made available across platforms and multiple chains. With the DripVerse platform, any user can access the utilities by choosing from what the developer ecosystem has created.

<CardGrid>
  <LinkCard title="Website" href="https://dripverse.org/" />
</CardGrid>

## Features

- **Tokens:** Support for ERC-721 token standard

- **Marketplace:** Buy and seel NFTs marketplace

- **NFTs Creation:** Allows creating NFTs collections

## Quickstart

<Steps>

1. Go to the Dripverse [platform](https://dripverse.org/).

2. Use the marketplace or create your own collection

</Steps>
