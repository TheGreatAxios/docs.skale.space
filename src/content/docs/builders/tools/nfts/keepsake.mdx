Keepsake is an NFT marketplace that incorporates unique features of the SKALE ecosystem, enhancing user navigation. Notably, it includes an embedded SKALE native bridging feature, allowing users to bridge various tokens between SKALE chains at no cost.

<CardGrid>
  <LinkCard title="Calyso Website" href="https://calypso.keepsake.gg/" />
  <LinkCard title="Nebula Website" href="https://nebula.keepsake.gg/" />
</CardGrid>

## Features

- **Payments:** Marketplace with supporting payments in USDC.

- **Bridging:** Integration with SKALE Native bridge to move USDC between chains.

- **Tokens:** Support both ERC-1155 and ERC-721 tokens.

## Quickstart

<Steps>

1. Go to the Keepsake marketplace. Select the Calypso or Nebula one.

2. Connect a wallet.

3. Start using the marketplace.

</Steps>
