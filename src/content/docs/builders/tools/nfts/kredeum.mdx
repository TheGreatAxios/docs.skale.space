## Features

- **IPFS:** Allows easy image stporing on IPFS

- **Factory:** Easy to use NFT factory 

- **Wordpress:** Criati

## Quickstart

<Steps>

1. Head to the Kredeum [platform](https://app.kredeum.com/).

2. Connect a wallet.

3. Click on add to mint a NFT or add a custom collection.

</Steps>
