## Features

- **REST API:** Snag offers a robust API suite to help you manage loyalty rules, track user interactions, and generate meaningful rewards.

- **SDK (JS/TS):** Snag offers an SDK for typesafe interaction with our APIs from server-side Typescript or Javascript.

- **Compatibility:** The whitelabel marketplace has compatability with multiple chains one of which SKALE Nebula is included.

## Quickstart

<Steps>

1. Head to the Snag [dashboard](https://admin.snagsolutions.io/admin).

2. Setup a project and connect a wallet.

3. Under **contracts** -> **Add New Contract** select SKALE Nebula

4. Customise your marketplace according to your preferances.

</Steps>
