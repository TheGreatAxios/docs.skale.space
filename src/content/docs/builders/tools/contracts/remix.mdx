Click this [link](https://remix.ethereum.org/?#code=Ly8gU1BEWC1MaWNlbnNlLUlkZW50aWZpZXI6IE1JVApwcmFnbWEgc29saWRpdHkgXjAuOC4yNDsKCgpjb250cmFjdCBIZWxsb1NLQUxFIHsKICAgIHN0cmluZyBwdWJsaWMgc3RvcmVkTmFtZTsKCiAgICBmdW5jdGlvbiBzZXROYW1lKHN0cmluZyBtZW1vcnkgbmFtZSkgcHVibGljIHsKICAgICAgICBzdG9yZWROYW1lID0gbmFtZTsKICAgIH0KCiAgICBmdW5jdGlvbiBzYXlIZWxsbygpIHB1YmxpYyB2aWV3IHJldHVybnMgKHN0cmluZyBtZW1vcnkpIHsKICAgICAgICByZXR1cm4gc3RyaW5nKGFiaS5lbmNvZGVQYWNrZWQoIkhlbGxvICIsIHN0b3JlZE5hbWUsICIhIikpOwogICAgfQp9) to load the following Hello World contract into the Remix IDE:

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.24;


contract HelloSKALE {
    string public storedName;

    function setName(string memory name) public {
        storedName = name;
    }

    function sayHello() public view returns (string memory) {
        return string(abi.encodePacked("Hello ", storedName, "!"));
    }
}
```

<Steps>

1. #### Contract Compilation 

    <Image src={remixCompile} alt="Remix Contract Compilation"/>

        **1.** Select the `Solidity compiler` tab
        
        **2.** Select the `Compile` button

    :::note[Developer Tip]
    On Advanced Configurations please select an EVM Version equal or lower to `Paris`
    :::

2. #### Wallet Setup

    <Image src={remixWallet} alt="Remix Contract Deployment"/>

        **1.** Select the `Deploy & run transactions` tab
        
        **2.** Under environment select `Injected Provider` option

        :::note[Note]
        If required please go to [SKALE Portal](https://testnet.portal.skale.space/chains) and connect your wallet to the SKALE Chain you want
        


3. #### Contract Deployment

    <Image src={remixBtnDeploy} alt="Remix Contract Deployment"/>
    Select the deploy button to have your contract available on the selected SKALE Chain



</Steps>
