## Implementation Example

<Steps>

1. ##### Get an API key

    Please visit developer console, select the [staging/testnet](https://staging.crossmint.com/signin?callbackUrl=/console) or [production/mainnet](https://www.crossmint.com/signin?callbackUrl=/console) environment and get your API Key.

2. ##### Integration

    ```javascript
        import fetch from "node-fetch";

        const body = {
            email: "<YOUR_USERS_EMAIL>",
            chain: "skale-nebula-testnet",
        };

        const response = await fetch(`https://staging.crossmint.com/api/v1-alpha1/wallets`, {
            method: "POST",
            headers: {
                "X-API-KEY": "<YOUR_API_KEY>",
                "content-type": "application/json",
            },
            body: JSON.stringify(body),
        });

        const wallet = await response.json();
    ```
