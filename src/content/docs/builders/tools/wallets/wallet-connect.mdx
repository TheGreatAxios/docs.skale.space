The Wallet Connect is an open source protocol to link dApps to mobile wallets using deep linking, and allows you to integrate your dApp with SKALE.

## Features

- **Multi-Wallet Compatibility:** Enables seamless connection between dApps and various wallets via QR code or deep linking.  

- **Cross-Platform Support:** Works across mobile, web, and desktop environments for flexible usage.
