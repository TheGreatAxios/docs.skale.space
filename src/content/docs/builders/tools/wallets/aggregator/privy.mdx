## Implementation Example

<Steps>

1. ##### Get an API ID

    - Login to [Privy dashboard](https://dashboard.privy.io/)
    - Click on `New App` on the Applications page
    - Click on the Settings button on the left pane. Under the Basic tab, you'll find the API keys.

2. ##### Select the Privy SDK

    Head to [Privy SDK](https://docs.privy.io/#explore-by-sdk) section and select which ones suits your better.

3. ##### Package Install

    <Tabs>
    <TabItem label="React">
        ```shell
        npm i @privy-io/react-auth
        ```
    </TabItem>
    <TabItem label="Unity">
        Please request the SDK access by reaching out the team through the following email: `sales@privy.io`
    </TabItem>
    <TabItem label="Swift">
        - Open `Xcode` -> `File` -> `Add Package Dependencies...`
        - Enter `https://github.com/privy-io/privy-ios`
        - Add `import PrivySDK` on top of the file
    </TabItem>
    <TabItem label="Expo">
        **Core Install**

        ```shell
        npx expo install expo-application expo-constants expo-web-browser expo-linking expo-secure-store react-native-webview expo-crypto @privy-io/expo-native-extensions @privy-io/expo
        ```

        **Polyfills Install**
        ```shell
        npm i fast-text-encoding react-native-get-random-values @ethersproject/shims
        ```

        <Aside>
            If your app uses the `Expo bare workflow` please import:

            ```shell
            npx pod install
            ```
        </Aside>
    </TabItem>
    </Tabs>

4. ##### Integration

    <Tabs>
    <TabItem label="React">

        Check the [Privy Swift quickstart](https://docs.privy.io/guide/swift/setup/).

    <TabItem label="Expo">
        Check the [Privy Expo quickstart](https://docs.privy.io/guide/expo/quickstart).
    </TabItem>
    </Tabs>

</Steps>
