## Features

- **Privacy-Focused Wallet:** Enables secure and private key management for users.  
- **Customizable Authentication:** Offers developers flexibility in integrating wallets with various user flows.  
- **SKALE Chains:** All SKALE Hubs are compatible with the Privy Wallets.

## Quickstart

<Steps>

1. Install the package:
    ```shell
    npm i @privy-io/react-auth
    ```
2. Get an API Key on the [Privy dashboard](https://dashboard.privy.io/).

3. Check the [Privy quickstart](https://docs.privy.io/guide/react/quickstart).

</Steps>
