Meson Finance is a stable coin bridge that connects SKALE chains with more than 35 different chains, enabling to move different stable coins (USDC and USDT) and ETH in and out of SKALE network.

## Features

- **Multiple networks:** SKALE Chains connected with a lot of diferent networks. 

- **Tokens:** Multiple tokens support -> **USDC**, **USDT**, **ETC** and **WBTC**. 

- **Embeded Integrations:** There're different solutions available to integrate the plaform in seamless manner.

- **Smart Contract Interaction:** You can directly interact with the smart contracts for bridging, instead of using the solutions provided by the Meson team. This can be helpful for dApps that use features like custodial wallets.

## Quickstart

<Steps>

1. Visit https://meson.fi/.

2. Connect your wallet.

3. Select the destination chain, origin chain, token to bridge and amount.

4. Bridge.

</Steps>
