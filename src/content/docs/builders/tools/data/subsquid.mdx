Subsquid is a query node framework that can be applied to SKALE Chains.

Developers can use Subsquid to extract SKALE Nebula chain data and process EVM logs in one project and serve this data on a GraphQL endpoint.

## Features

- **Flexible Data Indexing:** Offers customizable indexing pipelines to extract and transform blockchain data tailored to your specific needs.

- **High-Performance Querying:** Provides fast and efficient GraphQL APIs for querying large-scale blockchain data with ease.

- **Data Hosting:** Subsquid hosts data on belhalf of the developer.

- **SKALE Nebula Hub:** Subsquid is available to be used on SKALE Nebula both mainnet and testnet.
