Dune serves as an incredibly flexible web3 analytics tool and source of truth, aligning with the SKALE Network approach to unveil the formerly hidden web3 landscape by showcasing previously web2-exclusive data on the Dune platform.

<CardGrid>
  <LinkCard title="Website" href="https://dune.com/skale/analytics" />
  <LinkCard title="Documentation" href="https://docs.dune.com/home" />
</CardGrid>

## Features

- **Custom Query Creation:** Allows users to write SQL queries on blockchain data to extract insights and create custom reports.

- **Interactive Dashboards:** Provides tools to build and share real-time, interactive dashboards for data visualization.

- **SKALE Network Data:** All SKALE Network data regarding chains management, nodes, validators, staking and SKALE Chains can be checked on Dune.

## Quickstart

<Steps>

1. Visit [Dune SKALE page](https://dune.com/skale).

2. Select the SKALE dashboards you would like to know about:

	- [Main](https://dune.com/skale/analytics)
	- [Validators](https://dune.com/skale/validator)
	- [Delegators](https://dune.com/skale/delegator)
	- [SKALE Multi-chain](https://dune.com/skale/statistics)
	- [SKALE Signle-chain](https://dune.com/skale/schain)
	- [Super Node](https://dune.com/skale/node)

3. Use any query of the dashboards or use the available API to access the query data externaly.

</Steps>
