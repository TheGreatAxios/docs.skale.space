## SKALE Network Overview

SKALE is a network of Layer-1 blockchains offering zero-gas fees and pooled security.
SKALE also offers an additional security anchor by having key management contracts, the default
bridge, and the network's utility token all on the Ethereum Mainnet.

You can learn more about SKALE [here](/learn/about-skale-network).
